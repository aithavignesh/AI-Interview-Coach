# 🤖 AI Interview Coach

An AI-powered interview preparation platform built using React, FastAPI, PostgreSQL, and Google's Gemini AI.

## Features

- AI Mock Interview
- Resume Analysis
- Coding Interview
- Company-specific Interview
- JWT Authentication
- PostgreSQL Database
- Dashboard & Analytics

## Tech Stack

- React
- FastAPI
- PostgreSQL
- SQLAlchemy
- Gemini API
- Tailwind CSS
- Monaco Editor
